I'm Chris Schroedinger. You'll find that name on a few things — a
[GitHub profile](https://github.com/ChrisSchroedinger) with more repos
than stars, an open-source dictation tool called
[AgentWhisper](https://chrisschroedinger.github.io/AgentWhisper/), and
now this blog.

Right now I'm working with AI agents — building them, wiring them
together, figuring out what they're actually good at and where they
fall apart. AgentWhisper is the main project: push-to-talk dictation
for Linux, fully local, because every voice product I could find either
sent audio to a cloud or skipped Linux entirely. So I built the one
I wanted.

That project opened a door. Once you have voice-in for an agent, you
want the rest of the stack — memory, orchestration, computer control,
the whole pipeline. I've been building that too. This blog is where
I'll write about what works, what breaks, and what I learn along
the way.

There's no grand launch, no newsletter, no carefully engineered funnel.
Just a repo, a markdown file, a git push. If that sounds like your
kind of thing, pull up a chair.
