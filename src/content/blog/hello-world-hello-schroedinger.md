I'm Chris Schroedinger.

Right now I'm working with AI agents — building them, wiring them
together, figuring out what they're actually good at and where they
fall apart. Building tools and processes.
AgentWhisper is the first project: push-to-talk dictation
for Linux, fully local, because every voice product I could find 
skipped Linux entirely. So I let AI built the one I wanted.

That project opened a door. Once you have voice-in for an agent, you
want the rest of the stack — memory, orchestration, computer control,
the whole pipeline. I've been building that too. This blog is where
I'll write about what works, what breaks, and what I learn along
the way. And all the tools I manage to write with AI.

There's no grand launch, no newsletter, no funnel.
Just a repo, a markdown file, a git push. If that sounds like your
kind of thing, get some hot tea and pull up a chair.
